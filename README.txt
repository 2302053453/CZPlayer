我也在Qt吧里面发布了CZPlayer欢迎光顾，CZPlayer1.0用的播放引擎是phonon，CZPlayer2.0用的播放引擎是Multimedia,我最后改成Multimedia的主要原因是我想实现频谱显示，CZPlayer的主要功能有
1,：支持多种音乐格式播放（第二个版本只支持wav格式，因为Multimedia是一个比较底层的播放引擎，所以底层我只实现了wav格式的播放）
2：一台电脑只能开启一个实例（只能启动一个CZPlayer和酷狗，qq音乐一样）
3：实现全局热键
4：最小化到托盘
5：数据库功能，主要是保存音乐信息
6：人性化的配置
7：实时显示歌词，下载歌词，下载专辑图片
8：下载歌曲
9：搜索歌曲
10：迷你界面
11：频谱显示，波形显示
12：还有一些功能，就不详细的列出来了
